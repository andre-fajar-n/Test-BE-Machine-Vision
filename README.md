# Test-BE-Machine-Vision
Test skill for position Backend Engineer in Machine Vision

# Prerequisites
1. Node v18.15.0
2. npm 9.5.0
3. MySQL 8.0.32-0

# Setup Project
1. Copy .env.example to .env using this
``` bash
cp .env.example .env
```
2. Change value in the env variable based on your environment
3. Install all package
``` bash
npm install
```
4. Run the project
``` bash
npm run start
```
